
## 环境准备

- Node.js 10.15.1+
- yarn >= 1.5

```bash
$ nvm ls-remote --lts
$ nvm install v10.22.0
$ nvm use v10.22.0
$ nvm current
v10.22.0
```

- 详见 [nvm 管理 Node 版本](http://kail.xyz/JavaScript/docs/Npm/NVM/)


## 常用命令

```bash
# 安装依赖
$ yarn install

# 开启服务
$ yarn start

# 部署
$ GIT_USER=kailbin USE_SSH=false yarn deploy
```
